# 行为准则

[English](CODE_OF_CONDUCT.md)

本项目采用 Contributor Covenant 行为准则 3.0，并针对 agent-infra 社区做了适配。

## 我们的承诺

我们承诺让社区对所有人保持友好、安全且公平。

我们致力于营造一种尊重并促进每个人尊严、权利和贡献的环境，不因种族、族裔、种姓、肤色、年龄、身体特征、神经多样性、残障、性别、生理性别或社会性别、性别认同或表达、性取向、语言、哲学或宗教、民族或社会出身、社会经济地位、教育程度或其他身份状态而区别对待。

所有善意参与并遵守本行为准则的人，都享有同等的参与权利。

## 鼓励的行为

我们承认不同地区和文化中的社会规范可能存在差异，但所有成员都应努力满足本社区对积极行为的期望。我们也理解，受文化、背景或母语影响，我们的话语和行为可能被他人以不同于本意的方式理解。

基于这些考虑，我们同意以审慎、尊重的方式彼此互动，并采取体现共同价值观的行为，包括：

1. 尊重社区目标、社区活动和协作方式。
2. 以友善和诚实的态度与他人互动。
3. 尊重不同观点和经历。
4. 对自己的行为和贡献负责。
5. 得体地提出并接受建设性反馈。
6. 当造成伤害时，愿意承担修复责任。
7. 采取其他有助于维护社区健康发展的行为。

## 受限制的行为

我们同意在社区中限制以下行为。实施、威胁实施或宣扬这些行为，均视为违反本行为准则。

1. 骚扰：违反他人明确表达的边界，或在对方明确要求停止后继续进行不必要的个人关注。
2. 人身攻击：针对社区成员或群体发表侮辱、贬低或带有蔑视意味的评论。
3. 刻板印象或歧视：基于不可改变的身份或特征来定义他人的人格或行为。
4. 性化行为：在社区上下文或目的下通常会被认为不适当亲密的行为。
5. 违反保密义务：未经许可分享或使用他人的个人或私密信息。
6. 危害安全：导致、鼓励或威胁对任何个人或群体施加暴力或其他伤害。
7. 其他威胁社区健康发展的行为。

### 其他限制

1. 误导性身份：冒充他人，或通过伪装身份规避执行措施。
2. 未注明来源：未正确标注所贡献内容的来源。
3. 宣传材料：以不符合社区规范的方式分享营销或其他商业内容。
4. 不负责任的沟通：以不负责任的方式呈现、链接或描述任何其他受限制的行为。

## 报告问题

即使社区成员都在尽力协作，成员之间仍可能出现紧张关系。并非所有冲突都构成行为准则违规；本行为准则也旨在强化鼓励的行为和社区规范，从而帮助避免冲突并减少伤害。

如果确实发生事件，请及时并私下报告。你可以通过电子邮件联系项目维护者：<codecaster365@outlook.com>。如果报告涉及该维护者，请使用 GitHub 平台举报工具，或联系 `fitlab-ai` 组织中列出的其他所有者。

报告应包含相关上下文、链接、截图、时间点、涉及账号，以及需要主持人考虑的任何安全顾虑。请不要在公开 Issue、Pull Request 或 Discussion 中发布报告内容。

社区主持人会严肃对待违规报告，并尽力及时回应。主持人会在适当情况下通过审阅消息、日志、录屏或证人说明来调查报告。主持人会在优先保障安全、隐私和保密性的前提下，尽可能保持调查和执行措施透明。

为了维护这些价值，执行措施通常会私下向相关当事人进行；但在各方共同认可的解决方案中，也可能包含向整个社区沟通的部分。

## 处理和修复伤害

如果社区主持人调查后确认本行为准则被违反，可以根据事件对相关个人和整个社区造成的影响，使用以下执行阶梯来判断如何修复伤害。根据违规严重程度，可以跳过较低层级的措施。

1. 警告
   1. 事件：涉及单个事件或一系列事件的违规。
   2. 后果：社区主持人发出私下的书面警告。
   3. 修复：修复方式可以包括私下书面道歉、承认责任，以及进一步澄清行为期望。
2. 临时限制活动
   1. 事件：曾被警告后再次发生违规，或首次发生较严重违规。
   2. 后果：发出私下书面警告，并设置有时限的冷静期，以强调事件严肃性，并让相关社区成员有时间处理该事件。冷静期可以限制在特定沟通渠道或与特定社区成员的互动中。
   3. 修复：修复方式可以包括道歉、利用冷静期反思行为和影响，以及在冷静期结束后谨慎回到社区空间。
3. 临时暂停
   1. 事件：社区主持人尝试通过警告处理后仍反复发生违规，或发生单次严重违规。
   2. 后果：发出私下书面警告，并附带暂停结束后回归社区的条件。通常，临时暂停是为了让被暂停者有时间反思自己的行为以及可能的纠正措施。
   3. 修复：修复方式包括尊重暂停措施的精神，满足指定的回归条件，并在暂停解除后谨慎重新融入社区。
4. 永久禁入
   1. 事件：重复违反行为准则且其他阶梯措施未能解决，或违规严重到社区主持人认为无法在该成员继续参与的情况下保障社区安全。
   2. 后果：移除其访问所有社区空间、工具和沟通渠道的权限。通常，永久禁入应很少使用，应有充分理由，并且只应在其他修复措施无法改变行为时作为最终手段。
   3. 修复：在这种严重程度的情况下，不存在可行的修复方式。

本执行阶梯是指导原则。它不限制社区主持人基于社区最佳利益运用判断和酌处权。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人以官方身份在公开或其他空间代表社区时的行为。代表社区的例子包括使用官方电子邮件地址、通过官方社交媒体账号发布内容，或以指定代表身份参加线上或线下活动。
